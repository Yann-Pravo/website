I joined Block Earner to build their cryptocurrency web application from the ground up and to create their website. The project required a skilled team and close coordination between frontend and backend development.

My primary responsibility was to develop the web application and website, recruit and manage the frontend team, and ensure seamless integration with the backend through REST and WebSocket API calls. Additionally, I was tasked with developing core functionalities such as buying, selling, and decentralized finance (DeFi) features.

- Created the cryptocurrency web application using React.js, TypeScript, Axios, React Query, TailwindCSS, and other libraries
- Developed the website with Next.js for optimal performance and SEO
- Recruited and managed a talented frontend team, ensuring effective collaboration and high-quality code.
- Provided mentorship and guidance to team members, fostering a productive and positive work environment.
- Worked closely with the backend team to integrate REST and WebSocket API calls, ensuring smooth communication between frontend and backend.
- Coordinated with other departments to align development goals with business objectives.
- Led the team in developing essential platform functionalities, including buy, sell, and DeFi features, ensuring a seamless user experience.

Successfully delivered a robust and feature-rich cryptocurrency web application and website. The collaboration and management efforts led to a highly efficient development process and a high-quality final product that met the company's objectives and user needs.
